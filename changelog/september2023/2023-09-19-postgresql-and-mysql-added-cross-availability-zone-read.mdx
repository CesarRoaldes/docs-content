The Cross AZ Read Replica feature ensures your data being replicated in a dedicated read-only node, which is located in another AZ within the same Region as your main node.

For more information, refer to the <Link to=https://www.scaleway.com/en/docs/managed-databases/postgresql-and-mysql/how-to/manage-read-replica/>How to manage Read Replicas
</Link> documentation page.
